# DevOps Skills

Automation, webhook-driven workflows, and infrastructure management for CI/CD pipelines and operational tooling.

## Skills

| Skill | Description |
|-------|-------------|
| `webhook-subscriptions` | Manage webhook subscriptions for external event-driven automation |

## Webhook Patterns

- Event-driven automation via HTTP webhooks
- Subscription management (create, list, delete)
- Payload parsing and routing
- Retry logic and dead-letter handling

## CI/CD Integration

Connect Hermes to CI/CD systems for:
- Automated testing feedback
- Deployment notifications
- Infrastructure provisioning triggers
- Monitoring and alerting pipelines

## Automation Pipelines

Build multi-step automation workflows that:
1. Receive external events (webhooks, scheduled triggers)
2. Process and route based on content
3. Execute actions across platforms
4. Log results and handle failures
