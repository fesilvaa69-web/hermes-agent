# Red Teaming Skills

Security research and adversarial testing capabilities for evaluating AI systems, identifying vulnerabilities, and conducting controlled security research.

## Skills

| Skill | Description |
|-------|-------------|
| `godmode` | Advanced adversarial prompt testing and red teaming framework |

## Red Teaming Workflows

### Prompt Injection Testing
- Discover prompt injection vulnerabilities
- Test boundary conditions and edge cases
- Evaluate model behavior under adversarial inputs

### Security Research
- Identify potential attack surfaces in AI systems
- Test for information leakage
- Evaluate safety filter effectiveness

### Controlled Testing
- Systematic evaluation frameworks
- Documented test cases with expected outcomes
- Result analysis and vulnerability classification

## Ethical Guidelines

- Only test systems you have authorization to test
- Follow responsible disclosure practices
- Document findings securely
- Comply with applicable laws and regulations
