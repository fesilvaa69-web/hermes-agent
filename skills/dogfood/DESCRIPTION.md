# Dogfood Skills

Internal testing and self-evaluation skills for Hermes Agent development and quality assurance.

## Purpose

Dogfood skills are used to test Hermes Agent's own capabilities by running it against challenging, real-world tasks and measuring performance. This includes:
- Benchmarking new skills before release
- Regression testing after updates
- Quality assurance workflows
- Self-evaluation of agent capabilities
